MODELO 3D · Heydar Aliyev Center

Para que el visor 3D muestre el modelo de la maqueta:

1. Entra a https://www.printables.com/model/1296501-heydar-aliyev-center-3d-printable-architecture-mod
2. Inicia sesion (la descarga requiere cuenta gratuita).
3. Descarga el archivo STL.
4. Guardalo aqui con el nombre exacto: heydar.stl
   (ruta final: assets/models/heydar.stl)
5. Recarga la pagina.

Si el archivo no esta presente, el visor mostrara automaticamente
un modelo parametrico de respaldo (no es el mismo de la maqueta).

Formato aceptado: .stl (binario o ASCII)
